Enhanced Color Scale Generator

Enter a base color (level 400) to generate a complete color scale from 100 to 700. You can enter a HEX color with or without the # prefix, or use the color picker.

How the Color Scale Works

This tool generates two complete color scales:

  • Original Scale: Based on your selected base color (level 400)
  • Accessible Alternative: If your base color doesn't pass accessibility requirements for both black and white text, an alternative scale is shown with the closest accessible color

Each scale automatically calculates:

  • Lighter variants (100-300) with increasing lightness and slightly decreasing saturation
  • Darker variants (500-700) with decreasing lightness and increasing saturation

The lighter shades never reach pure white, and the darker shades never reach pure black, ensuring visual distinction at all levels.

Accessibility: The tool checks contrast against WCAG AA standards for:
  • Normal text (4.5:1 ratio required): Text smaller than 18pt, or smaller than 14pt if bold
  • Large text (3:1 ratio required): Text at least 18pt, or at least 14pt if bold
Please enter a valid HEX color code (e.g. #3791c6)
